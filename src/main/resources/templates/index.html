Codigo morse

El estandar para el envio de mensaje en codigo morse es el siguientes:

  • Un punto es la unidad minima.
  • Tres puntos equivalen a un trazo.
  • Un silencio equivalente a un punto es el separador entre cada punto o trazo.
  • Un silencio equivalente a tres puntos es el separador entre caracteres.
  • Un silencio equivalente a siete puntos es el separador entre palabras.

Velocidad del emisor

El estandar para determinar la velocidad del emisor es enviar un primer mensaje con la palabra "PARIS" (.--. .- .-. .. ...) escrita 5 veces. Al comenzar con el envio del mensaje se le exigira que ingrese este primer mensaje.
